DESCRIPTION

	data/datafiles/ is a directory that contains some sample files to test or use hydrocomp.py.  
	
	The data files include some output files from a hydrologic application (WATER) to compare with NWIS data files.
	
	The USGS NWIS data files were retrieved from: 
	
		http://waterdata.usgs.gov/nwis/rt

	WATER application reference:
		
		http://pubs.usgs.gov/sir/2009/5248/
